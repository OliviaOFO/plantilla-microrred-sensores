## 🎯 Objetivo del Proyecto
El objetivo del equipo es diseñar y justificar técnicamente una propuesta de red o microrred de sensores, haciendo un énfasis riguroso en cómo se resuelve el sistema de comunicaciones y las técnicas aplicadas para el tratamiento de las señales.

## 📂 Estructura del Repositorio
Su repositorio contiene los siguientes elementos esenciales:
* `README.md`: Este archivo con las instrucciones generales de entrega.
* `PLANTILLA_ENTREGA.md`: El documento base obligatorio que contiene los 13 puntos que deben desarrollar para la memoria técnica.
* `/planos_ubicacion`: Carpeta local donde deben guardar las imágenes de los planos en planta/alzado.
* `/specs`: Carpeta local destinada a almacenar los PDFs o documentos de especificaciones de los fabricantes de los sensores elegidos.
